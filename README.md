# Awesome neovim configuration

## Purpose

the idea of this repo is to store my personal preferences of neovim configuration

## Dependencies

neovim nightly build (0.9+)
node (18.12+)
python (3.9+)
deno (1.29+)
fd (fd-find)
bat (batcat)
fzf
cargo
rustup
make

## How to install

just clone this repo in `~/.config/` directory
